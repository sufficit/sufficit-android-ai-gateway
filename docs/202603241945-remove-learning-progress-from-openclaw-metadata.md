## About

The Android app still computes local speaker learning progress for on-device UX, but this value is no longer sent to OpenClaw.

## Changes

- Removed `voiceLearningProgress` from outbound metadata generated by the Android gateway.
- Kept `sameSpeakerProbability` as the only voice-confidence signal sent for remote pre-agent decisions.
- Removed `learningProgress` from the pre-agent mirrored payload and workspace audit summary.

## Validation

- `./gradlew.bat :app:compileDebugKotlin`
- `npx eslint extensions/android/src/voice-pre-agent.js extensions/android/src/session-audit.js --fix`

## Notes

- Local Android UI can continue using learning progress internally without exposing it to OpenClaw or the web audit view.
